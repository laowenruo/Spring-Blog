# Spring-Blog
框架：Springboot

数据库持久层：Mybatis

文章评论插件：Valine

分页插件：PageHelper

后台UI框架：X-admin框架，即LayUI框架

数据库连接池：hikari

数据库：MySQL

日志：Log4J
